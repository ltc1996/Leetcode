78. 子集

- 中等

## 题目描述：
给定一组**不含重复元素**的整数数组 `nums`，返回该数组所有可能的子集（幂集）。

**说明**：解集不能包含重复的子集。

- 示例：
```
输入: nums = [1,2,3]
输出:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```


## 我的解答：
``` python
class Solution(object):
    # 用时24ms
    class Solution(object):
    def subsets(self, nums):
        """
        :type nums: List[int]
        :rtype: List[List[int]]
        """
        from copy import deepcopy as dp
        res = [[]]
        for i in nums:
            temp = dp(res)
            for j in temp:
                j.append(i)
            res += temp
        return res
```

## 最快解答
```python
class Solution(object):
    # 用时8ms
    def subsets(self, nums):
        res = [[]]
        for i in range(len(nums)):
            for subres in res[:]:
               res.append(subres+[nums[i]])
        return res
```
