0075. 颜色分类

- 中等

## 题目描述：
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。


## 注意：
不能使用代码库中的排序函数来解决这道题。

- 示例：

```
输入: [2,0,2,1,1,0]
输出: [0,0,1,1,2,2]
```

## 进阶：
- 一个直观的解决方案是使用计数排序的两趟扫描算法。首先，迭代计算出0、1 和 2 元素的个数，然后按照0、1、2的排序，重写当前数组。
- 你能想出一个仅使用常数空间的一趟扫描算法吗？


## 我的解答：
``` python
class Solution(object):
    # 用时24ms
    def sortColors(self, nums):
        """
        :type nums: List[int]
        :rtype: None Do not return anything, modify nums in-place instead.
        """
        # 双指针原地修改
        # fast 从 slow + 1 遍历到list尾
        i = 0
        while(i < len(nums) - 1):
            j = i + 1
            while(j <= len(nums) - 1):
                if(nums[i] > nums[j]):
                    nums[i], nums[j] = nums[j], nums[i]
                j += 1
            i += 1
```

```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        // 用时12ms
        int n = nums.size();
        int left = 0, curr = 0, right = n - 1;
        while(curr <= right){
            if(nums[curr] == 0){
                int temp = nums[left];
                nums[left] = nums[curr];
                nums[curr] = temp;
                left++;
                curr++;
            }
            else{
                if(nums[curr] == 1) curr++;
                else{
                    int temp = nums[right];
                    nums[right] = nums[curr];
                    nums[curr] = temp;
                    right--;
                }
            }           
        }
    }
};
```


## 最快解答：
``` python
class Solution(object):
    # 用时12ms
    def sortColors(self, nums):
        # 计数排序
        a = nums.count(0)
        b = nums.count(1)
        c = nums.count(2)
        for i in range(a):
            nums[i] = 0
        for i in range(a,a+b):
            nums[i] = 1
        for i in range(a+b, a+b+c):
            nums[i] = 2
```

```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        // 用时0ms
        int l = 0;
        int h = nums.size()-1;
        int k = 0;
        while (k <= h) {
            if (nums[k] == 0) {
                swap(nums[l++], nums[k]);
                k = max(l, k);
            }else if (nums[k] == 1) {
                k++;
            }else {
                swap(nums[h--], nums[k]);
            }
        }
        return;
    }
};
```
