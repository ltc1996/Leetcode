0019. 删除链表的倒数第N个节点

- 中等
给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

- 示例：
```
给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
```

## 说明：
给定的 n 保证是有效的。

## 进阶：
你能尝试使用一趟扫描实现吗？

## 我的解答：
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    # 用时44ms
    def removeNthFromEnd(self, head: ListNode, n: int) -> ListNode:
        fast = slow = head
        for _ in range(n):
            fast = fast.next
        if not fast:
            return head.next
    
        while fast.next:
            fast = fast.next
            slow = slow.next
        # print(slow.val, fast.val)
        # slow.val = slow.next.val
        slow.next = slow.next.next
        return head
```

## 最快解答：
```python
class Solution:
    # 用时12ms
    def removeNthFromEnd(self, head: ListNode, n: int) -> ListNode:
        root = ListNode(0)
        # 引入头节点
        root.next = head
        left = right = root
        while n:
            right = right.next
            n -= 1
        while right.next:
            right = right.next
            left = left.next
        left.next = left.next.next
        return root.next
```
