172. 阶乘后的0

## 题目描述：
给定一个整数 n，返回 n! 结果尾数中零的数量。

- 示例1：
```
输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
```

- 示例2：
```
输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```

说明：
> 算法的时间复杂度应为 O(log n) 。

## 我的解答：
``` python
class Solution(object):
    # 用时44ms
    def trailingZeroes(self, n):
        """
        :type n: int
        :rtype: int
        """
        # 所有的0都来自2*5
        # 而5的个数远少于2的个数
        # 判断1-n之间质因子5出现的次数
        result = 0
        while(True):
            result += n//5
            n = n//5
            if n == 0:
                return result
```

## 最快解答：
``` python
class Solution(object):
    # 用时32ms
    def trailingZeroes(self, n):
        # 思路一样，代码更精简
        res = 0
        while(n):
            res += n/5
            n /= 5
        return res
```
