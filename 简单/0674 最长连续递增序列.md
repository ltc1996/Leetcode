0674. 最长连续递增序列

- 简单
给定一个未经排序的整数数组，找到最长且**连续**的的递增序列。

- 示例1：
```
输入: [1,3,5,4,7]
输出: 3
解释: 最长连续递增序列是 [1,3,5], 长度为3。
尽管 [1,3,5,7] 也是升序的子序列, 但它不是连续的，因为5和7在原数组里被4隔开。 
```

- 示例2：
```
输入: [2,2,2,2,2]
输出: 1
解释: 最长连续递增序列是 [2], 长度为1。
```

**注意**: 数组长度不会超过10000。

## 我的解答：
```python
class Solution(object):
    # 用时80ms
    def findLengthOfLCIS(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if not nums:
            return 0
        count = 1
        res = 1
        for i in range(len(nums) - 1):
            if nums[i] < nums[i+1]:
                count += 1
            else:
                res = max(count, res)
                count = 1
        # 考虑到可能永远不会走进else，需取两次max
        return max(count, res)
```

```python
class Solution(object):
    # 用时64ms
    def findLengthOfLCIS(self, nums):
        if len(nums) == 0:
            return 0
        if len(nums) == 1:
            return 1
        max_len = 1
        count = 1
        for i in range(len(nums) - 1):
            if nums[i + 1] > nums[i]:
                count += 1
            else:
                count = 1
            # 放在if-else外面
            if max_len < count:
                max_len = count
        return max_len
```

		}

	}
