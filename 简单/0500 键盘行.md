500. 题目名称

- 简单

## 题目描述：
给定一个单词列表，只返回可以使用在键盘同一行的字母打印出来的单词。键盘如下图所示。

- 示例1：
```
输入: ["Hello", "Alaska", "Dad", "Peace"]
输出: ["Alaska", "Dad"]
```

注意：
1. 你可以重复使用键盘上同一字符。
2. 你可以假设输入的字符串将只包含字母。

## 我的解答：
``` python
class Solution(object):
    # 用时24ms
    def findWords(self, words):
        """
        :type words: List[str]
        :rtype: List[str]
        """
        # 定义满足同行的函数
        # 用集合的包含关系判断
        def inKB(word):
            keyboard = [set('qwertyuiop'), set('asdfghjkl'), set('zxcvbnm')]
            # 利用子集的包含关系，若A包含于B，则A∩B=A.
            # 或set(word).issubset(i)
            return any([set(word) & i == set(word) for i in keyboard])
        return [i for i in words if inKB(i.lower())]
```

                  ['a','s','d','f','g','h','j','k','l'],
                  ['z','x','c','v','b','n','m']]

            if str_lower[0] in keyboard[0]:

                tag=1
            else:
                tag=2
            for j in str_lower:
