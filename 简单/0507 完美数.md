507. 完美数

- 简单

## 题目描述：
对于一个 **正整数**，如果它和除了它自身以外的所有正因子之和相等，我们称它为“完美数”。

给定一个 **正整数** `n`， 如果他是完美数，返回 `True`，否则返回 `False`

- 示例：

```
输入: 28
输出: True
解释: 28 = 1 + 2 + 4 + 7 + 14
```

**注意**：

输入的数字 n 不会超过 100,000,000. (1e8)

## 我的解答：
``` python
class Solution(object):
    # 用时32ms
    def checkPerfectNumber(self, num):
        """
        :type num: int
        :rtype: bool
        """
        import math
        if num < 6:
            return False
        s = 1
        for i in range(2, 1+int(math.sqrt(num))):
            if num%i==0:
                s += (i+num/i)
        return s==num
```
