0349. 两个数组的交集

- 简单

## 题目描述：
给定两个数组，编写一个函数来计算它们的交集。

- 示例1：

```
输入: nums1 = [1,2,2,1], nums2 = [2,2]
输出: [2]
```

- 示例2：

```
输入: nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出: [9,4]
```

## 说明：
- 输出结果中的每个元素一定是唯一的。
- 我们可以不考虑输出结果的顺序。

## 我的解答：
``` python
class Solution(object):
    # 用时28ms
    def intersection(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: List[int]
        """
        return list(set(nums1)&set(nums2))
```

## 最快解答：
``` python
class Solution(object):
    # 用时20ms
    def intersection(self, nums1, nums2):
        return set(nums1) & set(nums2)
```
