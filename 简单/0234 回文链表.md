234. 回文链表

- 简单

## 题目描述：
请判断一个链表是否为回文链表。

- 示例1：
```
输入: 1->2
输出: false
```

- 示例2：
```
输入: 1->2->2->1
输出: true
```
**进阶**：你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题

## 我的解答：
``` python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution(object):
    # 用时152ms
    def isPalindrome(self, head):
        """
        :type head: ListNode
        :rtype: bool
        """
        res = []
        while head:
            res.append(head.val)
            head = head.next
        return res==res[::-1]
        # 用if-else分别return可提速
        # 用时96ms
        # if res==res[::-1]:
        #     return True
        # else:
        #     return False
```

## 最快解答：
``` python
class Solution(object):
    # 用时68ms
    def isPalindrome(self, head):
        rev = None
        slow = fast = head
        while fast and fast.next:
            fast = fast.next.next
            rev, rev.next, slow = slow, rev, slow.next
        if fast:
            slow = slow.next
        while rev and rev.val == slow.val:
            slow = slow.next
            rev = rev.next
        return not rev
```
