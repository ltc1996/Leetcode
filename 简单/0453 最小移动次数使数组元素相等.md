0453. 最小移动次数使数组元素相等

- 简单

给定一个长度为 *n* 的**非空**整数数组，找到让数组所有元素相等的最小移动次数。每次移动可以使 *n - 1* 个元素增加 1。

- 示例
```
输入:
[1,2,3]

输出:
3

解释:
只需要3次移动（注意每次移动会增加两个元素的值）：

[1,2,3]  =>  [2,3,3]  =>  [3,4,3]  =>  [4,4,4]
```

## 我的解答：
``` python
class Solution(object):
        # 用时304ms
        def minMoves(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # 等价与对一个元素-1直到所有元素相等
        m = min(nums)
        res = sum(nums) - m*len(nums)
        return res
```
