0065. 有效数字

- 困难

## 题目描述：
验证给定的字符串是否可以解释为十进制数字。

例如:
```
"0" => true
" 0.1 " => true
"abc" => false
"1 a" => false
"2e10" => true
" -90e3   " => true
" 1e" => false
"e3" => false
" 6e-1" => true
" 99e2.5 " => false
"53.5e93" => true
" --6 " => false
"-+3" => false
"95a54e53" => false
```

## 说明：
我们有意将问题陈述地比较模糊。在实现代码之前，你应当事先思考所有可能的情况。这里给出一份可能存在于有效十进制数字中的字符列表：

- 数字 0-9
- 指数 - "e"
- 正/负号 - "+"/"-"
- 小数点 - "."
当然，在输入中，这些字符的上下文也很重要。

## 我的解答：
``` python
class Solution:
    # 用时40ms
    def isNumber(self, s: str) -> bool:
        # space = False       # 是否出现过空格
        point = False         # 是否出现过小数点
        sign = False          # 正负号
        num = False           # 数字
        e = False             # e
        s = s.strip()         # 去掉s头尾的空格

        # left, right = 0, len(s) - 1
        # while left < len(s) and s[left] == ' ':
        #     left += 1
        # while right >= 0 and s[right] == ' ':
        #     right -= 1

        for c in s:         #[left: 1 + right]:
            if c == ' ':
                # s中段不能出现空格
                return False
            elif c in ('-', '+'):
                if sign or num or point:
                    # 正负号前有正负号 / 数字 / 小数点
                    return False
                else:
                    sign = True
            elif c == '.':
                if point:
                    return False
                else:
                    point = True
            elif c in ('1', '2', '3', '4', '5', '6', '7', '8', '9', '0'):
                # 出现数字
                num = True
            elif c == 'e':
                if not num:
                    # e之前无数字
                    return False
                else:
                    if e:   # 已出现过e?
                        return False
                    # elif point:
                    #     print(4)
                    #     return False
                    else:       # 全部重置为False
                        e = True
                        num = False
                        sign = False
                        point = False
            else:       # 出现其他字符
                return False
        # 出现过e后, 不能出现小数点, 必须出现数字
        if e:   
            if not num:
                return False
            if point:
                return False
        return num
```
